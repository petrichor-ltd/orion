用 EEG 理解香氣如何影響身心狀態。

Orion 結合 Muse 腦波擷取、引導式嗅覺測試與基準化 SRI/ARI 演算法，客觀量化香氣對放鬆與睡眠準備度的變化。

從設定到洞察，四步完成研究流程

Orion 讓受測者保持平靜，同時保留可重複、可追蹤、可分析的研究資料結構。

同步雲端

完成的 session 透過 API Key 驗證後上傳至 FastAPI 與 PostgreSQL，保存長期研究資料。

以腦波資料支持香氣研究

SRI 以 Alpha/Theta 放鬆頻帶相對於 Beta 警覺頻帶的變化為核心，並將每個香氣試驗與受測者自己的無香氣基準比較。

  • 擷取 Delta、Theta、Alpha、Beta、Gamma 等頻帶特徵。
  • 保留 artifact、樣本數與資料品質狀態，方便研究回溯。
  • 雲端 schema 可支援 session 檢視與後續 ML 資料匯出。
